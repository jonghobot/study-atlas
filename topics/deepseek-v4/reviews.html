Analysis

기술·제품·시장 관점 분석

DeepSeek V4의 의미는 성능표 하나가 아니라, 장문맥 에이전트 인프라의 비용 구조를 바꾸려는 시도에 있습니다.

1. 기술 관점

핵심 주장
1M context는 capacity이고, V4의 진짜 기여는 그 context를 감당하는 attention/KV cache 효율입니다.
구조
Compressed Sparse Attention과 Heavily Compressed Attention 계열을 섞어 긴 문맥의 계산량과 메모리 사용량을 낮춥니다.
효과
V3.2 대비 per-token FLOPs와 KV cache 부담을 크게 낮췄다고 설명됩니다.
주의점
긴 context가 항상 좋은 답을 보장하지는 않습니다. retrieval, memory 정리, tool result 압축 설계가 여전히 중요합니다.

2. 제품 관점

API 호환성
OpenAI ChatCompletions와 Anthropic 형식을 모두 지원해 기존 agent stack에 붙이기 쉽습니다.
가격 전략
V4-Flash는 매우 공격적인 가격으로 대량 처리와 라우팅 모델 역할을 노립니다.
모드 전략
Thinking/Non-Thinking을 같은 모델군 안에서 제공해 속도와 reasoning 품질을 선택하게 합니다.
마이그레이션
기존 deepseek-chat/reasoner는 V4-Flash 기반으로 흡수되고, 이후 폐기 예정입니다.
