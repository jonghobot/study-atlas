DeepSeek V4 관련 연구 리뷰

목록에서는 핵심만 빠르게 보고, 각 카드로 들어가면 문제의식·방법·기여·DeepSeek V4와의 연결·한계까지 자세히 읽을 수 있습니다.

MoE

DeepSeekMoE: Towards Ultimate Expert Specialization in Mixture-of-Experts Language Models

DeepSeek 계열 MoE 설계의 핵심 배경. expert를 더 잘 분화시키면서 token당 계산량을 낮추려는 연구입니다.

상세 리뷰 읽기

DSA

DeepSeek-V3.2 / DeepSeek Sparse Attention

V4의 CSA/HCA hybrid attention을 이해하기 위한 직접 선행. dense attention 비용을 줄이는 DeepSeek 계열 sparse attention 흐름입니다.

상세 리뷰 읽기

reasoning

DeepSeek-R1 계열 Reasoning Model

DeepSeek가 reasoning model로 시장에 강한 인상을 남긴 계열. V4의 Thinking/Non-Thinking 모드를 이해하는 배경입니다.

상세 리뷰 읽기

decoding

Multi-Token Prediction / MTP

한 번에 다음 여러 토큰을 예측하는 학습/추론 전략. V4의 장문맥 자체보다는 decoding throughput과 speculative decoding 가능성에 연결됩니다.

상세 리뷰 읽기

attention kernel

FlashAttention: Fast and Memory-Efficient Exact Attention with IO-Awareness

attention을 GPU에서 더 빠르고 메모리 효율적으로 계산하는 대표 연구. V4와는 kernel/implementation 관점에서 비교됩니다.

상세 리뷰 읽기

context extension

YaRN: Efficient Context Window Extension of Large Language Models

RoPE 기반 LLM의 context window를 늘리는 대표 연구. V4와 비교하면 “길이 확장”과 “긴 길이의 추론 비용” 차이가 드러납니다.

상세 리뷰 읽기

fine-tuning

LongLoRA: Efficient Fine-tuning of Long-Context Large Language Models

긴 context 능력을 저렴하게 fine-tuning하는 연구. V4를 도메인 특화 agent로 적응시키는 문제와 연결됩니다.

상세 리뷰 읽기

systems

Ring Attention with Blockwise Transformers for Near-Infinite Context

아주 긴 context를 시스템 병렬화로 처리하는 연구. V4의 모델 구조적 접근과 대비됩니다.

상세 리뷰 읽기

streaming

StreamingLLM: Efficient Streaming Language Models with Attention Sinks

무한히 이어지는 입력 스트림에서 어떤 token을 보존해야 하는지 다루는 연구. agent memory 설계와 연결됩니다.

상세 리뷰 읽기

KV cache

KV Cache Compression for Long-Context LLM Inference

긴 context inference의 실제 병목인 KV cache 메모리를 줄이는 연구군. DeepSeek V4의 핵심 주장과 가장 직접적으로 맞닿아 있습니다.

상세 리뷰 읽기

agent eval

SWE-bench / Repo-level Coding Agent Benchmarks

LLM이 실제 GitHub issue를 고치고 테스트를 통과하는지 보는 평가 흐름. V4의 agentic coding 주장을 검증하는 기준입니다.
