V4-Pro

1.6T total / 49B active 파라미터. 고난도 reasoning, coding, 장기 에이전트 작업을 겨냥한 플래그십 모델.

V4-Flash

284B total / 13B active 파라미터. 빠르고 저렴한 실사용 모델. 채팅, 요약, 라우팅, 간단한 agent task에 적합.

핵심 변화

CSA/HCA 계열 hybrid attention과 KV cache 압축으로 긴 컨텍스트 추론 비용을 낮춘 점이 핵심입니다. DeepSeekMoE, DSA, MTP, LLM parallelism, YaRN, LongLoRA, Ring Attention, StreamingLLM, FlashAttention 같은 관련 연구 흐름과 함께 읽어야 합니다.
