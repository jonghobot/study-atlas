문제의식
piano performance를 audio, symbolic, text modality로 묶은 dataset 방향입니다.
방법
연주 데이터에 텍스트 설명과 symbolic/audio 표현을 함께 연결합니다.
기여
text description과 symbolic/audio를 함께 쓰는 controllable generation 기반을 제공합니다.
Pop2Piano 연결
text prompt 기반 piano cover control 또는 평가 설명 데이터로 확장 가능합니다.

논문별 퀴즈

단답 주관식

  1. PIAST가 묶는 modality는 무엇인가?

    audio, symbolic, text modality를 함께 묶는다.

  2. text modality가 piano generation에 유용한 이유는?

    텍스트 설명으로 style, 난이도, 표현 의도를 조건화할 수 있기 때문이다.

  3. PIAST와 pop-to-cover paired setting의 차이는 무엇인가?

    PIAST는 piano performance 중심이고 pop song과 piano cover pair와는 직접 같지 않다.

  4. PIAST를 Pop2Piano 후속 연구에 활용하려면 무엇이 필요한가?

    도메인 transfer, prompt/control 설계, pop-cover pair와의 연결이 필요하다.

객관식

  1. PIAST의 특징은?

    multimodal piano dataset

  2. 포함 modality로 적절한 것은?

    audio, symbolic, text

  3. text가 유용한 이유는?

    style/표현 의도 조건화

  4. Pop2Piano와의 차이는?

    piano performance 중심 vs pop-cover pair
